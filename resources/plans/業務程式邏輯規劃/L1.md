# L1：模式選擇（商家層）

> **角色：商家。** 此層用於商家在開始營業前 / 暫停期間 / 切換用途時配置機器人狀態。**無語音、無機器人動作**，純終端互動。

---

## 入口（從何處進來）

| 來源 | 條件 |
|---|---|
| 程式啟動 | `python3.11 myProgram/myProgram.py` 後直接到此層 |
| L1 客服模式 | 印完電話後自動回此選單 |
| L1 待機模式 | 商家按指定鍵退出待機後 |
| L2 / L3 / L4 / L5 | 套用「[L0 子例程 A：回 L1 叫賣](L0_共通.md#子例程-a回-l1-叫賣)」，落點為 L1 **叫賣模式運行中**（非選單）|

---

## 進入時動作

- **[終端]** 印模式選擇提示：
  ```
  ====================================
  請選擇模式：
    1 — 叫賣模式
    2 — 待機模式
    3 — 客服模式
  （任何時刻按 q 直接退出程式）
  ====================================
  > 
  ```

---

## 邏輯鏈路

### 鏈路 A — 客服模式（商家查電話）

- **trigger**：商家終端輸入 `3`
- **進入動作**：[終端] 印商家客服電話（從 [L0 客服電話](L0_共通.md#-客服電話placeholder) 取）
- **轉到**：自動回 L1 選單（無等待 / 無確認）

> **客服模式從 L1 進入的返回語義**：印完電話**立即**回 L1 選單。對比 L4 客服模式（不自動返回、需手動選「退出 / 繼續」）— 因為 L1 沒有「進行中的交易」可保留。

### 鏈路 B — 待機模式（商家暫停）

- **trigger**：商家終端輸入 `2`
- **進入動作**：
  - [終端] 印「進入待機模式，按 `r` + Enter 回主選單」
  - 程式保持在此狀態，**不主動做任何事**（不叫賣、不偵測 OpenCV、不發語音）
- **轉到**：商家按 `r` + Enter → 回 L1 選單
- **`q` 全域退出仍生效**（待機期間商家按 `q` 仍可立即終止程式，見 [L0 全域規則](L0_共通.md#-全域規則)）

> **OpenCV 行為（2026-05-25 修訂）**：待機模式期間 OpenCV **完全關閉**（不偵測 / 不觸發 L2）。商家暫停 = 完全暫停。商家按 `r` 回主選單後 **OpenCV 維持關閉** — L1 主選單預設不偵測，只有商家選 `1` 進叫賣模式才啟動 OpenCV（一致原則：「商家未明確選叫賣前不偵測」）。

### 鏈路 C — 叫賣模式（吸引顧客 + OpenCV 觸發進 L2 的入口）

- **trigger**：商家終端輸入 `1`
- **進入動作**：
  - [終端] 印「進入叫賣模式」
  - 開始輪播：每 `HAWK_INTERVAL` 秒（= 12s）播一組叫賣術語，依 6 組 mod 6 輪流（見 [L0 6 組叫賣術語](L0_共通.md#-6-組叫賣術語)）
  - **同時** OpenCV 開始偵測相機範圍是否有人
- **轉到**：OpenCV 偵測到人 → **L2**

> **叫賣模式運行中按鍵規則（已敲定）**：只接受 `q` 全域退出；按 `1`/`2`/`3` 或其他鍵不切換模式（避免叫賣中誤切回選單導致 OpenCV 中斷）。如需切回選單，按 `q` 退出後重啟程式。

> **OpenCV 觸發定義（已敲定 2026-05-24）**：偵測到「相機框內持續有人 ≥ `OPENCV_DWELL`（= 1.5）秒」才認可為有效觸發。瞬時偵測（路人經過、商家走過、影子變化）會被過濾。

> **叫賣模式啟動行為（已敲定 2026-05-24）**：啟動時**不**套用「回 L1 子例程」（即第一輪不跳過）— 因為這是商家主動「開店」，不是顧客剛離開。子例程 A 的引用場景只有「L2-L5 結束後回 L1」。

## 出口列表（所有可能的 next state）

| 出口 | 觸發條件 |
|---|---|
| L1 自己（選單）| 客服印完電話 / 待機按 `r` |
| L1 叫賣模式運行中 | 鏈路 C 觸發（商家按 1）/ L2-L5 套子例程 A 後落點 |
| L2 | 叫賣模式運行中 + OpenCV 偵測到人 |
| 程式終止 | 全域 `q` |

---

## 全域行為（由 [L0 全域規則](L0_共通.md#-全域規則) 覆蓋）

- 任何狀態（選單 / 客服 / 待機 / 叫賣中）下，終端輸入 `q` → 立即退出程式
