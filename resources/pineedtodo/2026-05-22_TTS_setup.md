**建立日期：** 2026-05-22
**對應提交：** `fcbe360` — Add edge-tts (zh-TW-HsiaoChenNeural) for speak()

本次部署在程式碼層接入了 edge-tts，但 Pi 上還沒裝套件 / 設好音訊。請依下列順序在 Pi 終端執行。

完整參考：`/home/pi/Desktop/project_jiqiren` 同步後查 `resources/requirements/raspberry_pi_setup.md` 內的「語音輸出（TTS）— edge-tts」區段（**注意**：因為 `resources/` 在 `.gitignore`，這檔不會同步到 Pi，需要在 Windows 本機看）。

---

## Step 1：安裝套件

```bash
pip install edge-tts
sudo apt install -y mpg123
```

驗證：

```bash
which mpg123
python3 -c "import edge_tts; print(edge_tts.__version__)"
```

`which mpg123` 應印出 `/usr/bin/mpg123` 之類路徑；`python3 -c ...` 應印出版本號不報錯。

---

## Step 2：設定音訊輸出裝置

```bash
sudo raspi-config
```

選 → `System Options` → `Audio` → 依照你接的喇叭選 **HDMI** 或 **3.5mm jack（Headphones）**。
選完按 `Finish` 離開。

> 若 Pi 是接外接 USB 音效卡，可能要選 `USB Audio` 或在 alsamixer 內切卡。

---

## Step 3：調音量

```bash
alsamixer
```

操作鍵：
- 上下方向鍵：調 PCM 音量（建議 70–85%）
- 左右方向鍵：切換 channel（如果有多軌）
- `M`：靜音切換（看到 `MM` 表示靜音，要按 `M` 解除變 `OO`）
- `Esc`：離開

---

## Step 4：測試能否發聲

```bash
python3 -c "import edge_tts, asyncio; asyncio.run(edge_tts.Communicate('測試一下', 'zh-TW-HsiaoChenNeural').save('/tmp/t.mp3')); import subprocess; subprocess.run(['mpg123','-q','/tmp/t.mp3'])"
```

預期：聽到台灣女聲說「測試一下」。

### 故障排除

| 症狀 | 可能原因 | 排查 |
|---|---|---|
| `ModuleNotFoundError: edge_tts` | Step 1 沒裝或裝到別的 Python | `which python3` 確認、`pip show edge-tts` 看安裝位置 |
| `mpg123: command not found` | Step 1 沒裝 mpg123 | 重跑 `sudo apt install -y mpg123` |
| 跑完沒聲音 | 音訊輸出裝置選錯 / 靜音 / 音量 0 | 回 Step 2、Step 3 確認；`speaker-test -t wav -c 2 -l 1` 試系統內建測試聲 |
| `aiohttp.ClientError` 之類網路錯 | edge-tts 連不到 Microsoft 雲端 | 確認 Pi 有網路：`ping 8.8.8.8`、`curl -I https://speech.platform.bing.com` |

## Step 5：跑主程式聽看看

```bash
cd /home/pi/Desktop/project_jiqiren/myProgram
python3 myProgram.py
```

預期看到：

```
========================================
指令：y = 偵測到顧客，q = 退出
========================================
```

接著叫賣模式會自動循環，**從喇叭聽到台灣女聲**講「來喔！冰紅茶27元、刮刮樂180元，全場九折，走過路過不要錯過！」

按 `y` 進顧客模式 → 應聽到「歡迎光臨！冰紅茶27元、刮刮樂180元，全場九折。請問要什麼？」

輸入 `冰紅茶 一杯` Enter → 應聽到「好的，冰紅茶×1，共27元！請問還需要其他嗎？」

按 `q`（或 Ctrl+C）退出。
