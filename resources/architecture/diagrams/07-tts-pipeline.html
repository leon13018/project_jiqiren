所以呢 · TTS 三鐵則

  • ●內容定址快取：key＝SHA1(VOICE|rate|text)，任一合成參數變即自然失效；package-anchored（非 cwd）；執行期自我增長（動態句首播後永久免合成、斷網可播）。
  • ●1-deep prefetch：播放阻塞期間合成下一句進快取，句間靜默從一次 synth round-trip 降到趨近 0（單 thread 重疊、零新鎖、各句快取檔名互異）。
  • ●常駐 event loop 取代每句 asyncio.run；print 在 caller thread 保 SSH log 時序；ALSA drain 僅在還有下一句時 sleep 0.3s（防 mpg123 沖 ALSA buffer 截尾）。
